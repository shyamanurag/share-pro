# App Icons for PWA

For the PWA to work properly, you'll need to add the following icon sizes to this directory:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

These icons should be square and use the TradePaper branding (green background with "TP" or a trading symbol).
